# Distribution multi search
Сайт для линуксоидов. Вопросы, ответы и полезные материалы для большинства дистрибутивов

*Прототип*

## Запуск
1. Установите докер ([Windows](https://docs.docker.com/desktop/setup/install/windows-install/))
2. Перейдите в корень проекта (там где лежат файлы Dockerfile и docker-compose)
3. Введите команду в терминале - `docker compose up -d --build`
4. После того как проект сбилдится, перейдите в браузер по адресу http://127.0.0.1:3000
